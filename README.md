# 📊 Job Market Analysis Dashboard

## 📌 Overview

This project analyzes job market data to identify hiring trends, in-demand skills, and geographic distribution using Power BI.

## 🛠 Tools Used

* Power BI
* Python (for data cleaning)
* Pandas

## 📊 Key Insights

* Software Engineer roles dominate the job market
* Communication and problem-solving skills appear frequently
* Hiring is concentrated in major global cities
